ngWheel
============

AngularJS `ng-scroll` directive to bind *'scroll'* event.

### Usage:

```js
angular.module('app', [
    // ...
    'ngWheel'
]);
```

```html
    <div class="scrollable-element" ng-scroll="ctrl.onScroll($event)">
        scroll me!
    </div>
```

### Installing using npm

`npm i ngwheel`
